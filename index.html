Open-Minded

El presente proyecto pretende llevar a cabo un sitio web y una aplicación móvil nativa para Android y IOS, donde se administre la información demográfica de los estudiantes, para que la información se pueda consultar, eliminar, editar y buscar. Con el objetivo de organizar la información para el área administrativa de una universidad empleando estructuras de información.

Tecnologias usadas:
Caracteristicas:
En el inicio donde puedes realizar diferentes acciones como agregar un estudiante, ver a los estudiantes agregados, ver una alerta sobre la protección de datos personales e información adiccional sobre el proyecto. Una vez agregados, se podrán editar y visualizar la información del estudiante.
En este apartado podrás editar la información y tambien eliminar cualquier registro, antes de realizar dicha acción la aplicación muestra una alerta sobre la eliminación del estudiante.
